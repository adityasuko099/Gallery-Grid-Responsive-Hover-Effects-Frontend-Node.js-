# Tugas 3 - Responsive Gallery (Frontend Node.js)

##  Deskripsi
Proyek ini menampilkan gallery foto dengan layout grid responsif dan efek hover modern. Semua asset di-handle oleh Vite dan styling menggunakan Tailwind CSS.

##  Teknologi yang Digunakan
- Node.js
- Vite
- Tailwind CSS v3

##  Struktur Folder
src/
├ assets/
│ ├ img1.jpg
│ ├ img2.jpg
│ ├ ...
│ └ img9.jpg
├ style.css
└ main.js
index.html
tailwind.config.js
postcss.config.js



##  Cara Menjalankan Proyek
```bash
npm install
npm run dev
Buka link Local server pada browser:
http://localhost:5173/  (atau port lain sesuai terminal)
```
